### Eigen spel

Schrijf zelf een programma of een vraag-antwoord spelletje.

Gebruik in ieder geval:
- [x] Invoer: input()
- [x] Strings en getallen (int of float)
- [x] if / else
- [x] Geneste if-statements (if in een andere if)
- [x] Iets leuks dat je zelf bedenkt
